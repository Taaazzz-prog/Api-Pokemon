📖 Instructions d'Utilisation

⭐ Comment utiliser les Favoris :

1. Allez sur Pokédex ou Générations
2. Survolez une carte Pokémon → les boutons 🤍 et ⚖️ apparaissent
3. Cliquez sur 🤍 → le cœur devient ❤️ (ajouté aux favoris)
4. Cliquez "⭐ Favoris" dans le menu → modal avec tous vos favoris

🔄 Comment utiliser le Comparateur :

1. Cliquez sur ⚖️ sur plusieurs cartes → boutons deviennent ✓
2. Cliquez "🔄 Comparateur" dans le menu
3. Modal de comparaison s'ouvre avec graphiques et statistiques

📚 Comment utiliser l'Historique :

1. Utilisez la recherche avancée (bouton 🔍)
2. Faites plusieurs recherches
3. Cliquez "📚 Historique" → voir toutes vos recherches
